669. 换硬币

### **描述**

ENG

给出不同面额的硬币以及一个总金额. 写一个方法来计算给出的总金额可以换取的最少的硬币数量. 如果已有硬币的任意组合均无法与总金额面额相等, 那么返回 `-1`.

你可以假设每种硬币均有无数个
总金额不会超过10000
硬币的种类数不会超过500, 每种硬币的面额不会超过100

您在真实的面试中是否遇到过这个题？ 是

题目纠错

### **样例**

**样例1**

```
输入：
[1, 2, 5]
11
输出： 3
解释： 11 = 5 + 5 + 1
```

**样例2**

```
输入： 
[2]
3
输出： -1
```
